# Algorand Voting DApp

This project is a simple decentralized voting application built on Algorand.

## Project Idea
The goal of this project is to create a secure and transparent voting system using blockchain technology.

## Features Planned
- User voting interface
- Vote recording on Algorand
- Transparent and secure results
- Simple smart contract integration

## Purpose
This repository will be used throughout the course to build and maintain the Algorand project.
